- Aplicar os conhecimentos adquiridos em aula.

- Acompanhar minha evolução como desenvolvedor.

Ao longo do curso, serão utilizadas tecnologias como:

- JavaScript

- Git e GitHub
- Outras ferramentas e linguagens conforme o andamento do curso.
